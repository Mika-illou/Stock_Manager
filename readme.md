# Stock Manager 

Bienvenue dans Stock Manager, un projet final qui marque la fin de
notre parcours en HTML & CSS niveau débutant.\
Ce projet met en avant votre capacité à concevoir une interface moderne,
réactive et fonctionnelle tout en respectant un **guide de style
professionnel**.


## Description du Projet

Stock Manager est un tableau de bord moderne permettant la gestion
simple et visuelle du stock.\
Ce projet a été conçu pour :

-   mettre en pratique toutes les compétences HTML/CSS acquises,
-   respecter un système visuel cohérent** (couleurs, typographies,
    composants...),
-   offrir une interface professionnelle, structurée et intuitive,
-   montrer votre sens du détail, du design et de l'ergonomie.

## Objectifs du Projet

-   Parfaire et finaliser votre apprentissage HTML/CSS\
-   Mettre en pratique tous vos acquis\
-   Créer un dashboard moderne et responsive\
-   Maîtriser les composants : sidebar, navbar, cartes,statistiques...\
-   Respecter un guide de style strict inspiré des standards UX/UI actuels

## Fonctionnalités Clés

### 1. Navigation & Structure

-   Sidebar fixe avec icônes (Font Awesome)
-   Navigation claire et minimaliste
-   Zones distinctes : statistiques, liste, formulaires...

### 2. Cartes & Statistiques

-   Cartes dynamiques avec ombres et hover
-   Statistiques visuelles (stocks, alertes...)
-   Badges d'état (active, warning, error)

### 3. Gestion du Contenu

-   Formulaires stylisés
-   Champs avec focus animé
-   Modèles de cartes produits

### 4. Style Cohérent (Guide respecté)

-   Palette de couleurs complète
-   Système typographique hiérarchisé
-   Boutons : primaire, secondaire, danger
-   Ombres, arrondis, espacement standardisés

### 5. Responsive Design

-   Grilles 1--4 colonnes
-   Adaptation mobile automatique\
-   Sidebar rétractée en mobile\
-   Composants centrés en petit écran


## Technologies Utilisées
  Technologie et Rôle

  HTML: Structure du dashboard
  CSS: Design, layout, animations, responsive

  Flexbox / Grid: Mise en page moderne
  Font Awesome:Icônes
